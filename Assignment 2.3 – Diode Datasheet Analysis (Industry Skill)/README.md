# Diode Datasheet Analysis – 1N4007

## Diode Name
1N4007

## Manufacturer
Vishay Semiconductors

## Diode Type
Silicon Rectifier Diode

## Real-World Application
The 1N4007 is a staple component in AC to DC power conversion systems. Functioning as a rectifier, it ensures that electrical current moves in a single direction. This capability is vital for transforming alternating current (AC) into steady direct current (DC), which is required to power devices such as wall adapters, battery chargers, and various internal power supply modules.
