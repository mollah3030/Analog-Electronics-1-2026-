## Question 4(a) Thermal voltage at 25°C

Given:
- T(C) = 25 °C
- k = 1.38 × 10^-23 J/K
- q = 1.6 × 10^-19 C

Formula:
- T(K) = T(C) + 273
- Vt = (k * T) / q

Steps:
1) T = 25 + 273 = 298 K
2) Vt = (1.38 × 10^-23 * 298) / (1.6 × 10^-19)
3) Vt ≈ 0.0257 V = 25.7 mV

Final:
- Vt ≈ 25.7 mV


## Question 4(b) Diode current using diode equation

Given:
- Is = 40 nA = 40 × 10^-9 A
- n = 2
- Vt = 0.0257 V
- Vd = 0.5 V

Formula:
- Id = Is * ( e^(Vd / (n * Vt)) - 1 )

Steps:
1) n*Vt = 2 * 0.0257 = 0.0514
2) Vd/(n*Vt) = 0.5 / 0.0514 ≈ 9.73
3) e^(9.73) ≈ 1.68 × 10^4
4) Id ≈ 40 × 10^-9 * (1.68 × 10^4 - 1)
5) Id ≈ 6.7 × 10^-4 A

Final:
- Id ≈ 0.00067 A = 0.67 mA
